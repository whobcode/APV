## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Add your algorithm implementation
4. Include tests and documentation
5. Submit a Pull Request

## Adding New Algorithms

When adding a new pathfinding algorithm:
1. Create the implementation in `src/algorithms/`
2. Add visualization support in components
3. Include AI explanation prompts
4. Update the algorithm selector
